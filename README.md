SALES ANALYSTICS DASHBOARD (Excel)

This project contains an Excel Sales Analytics Dashboard created using multiple sheets, pivot tables, charts, and slicers.
It helps in understanding sales performance, customer behavior, product demand, and seasonal trends.

Dashboard Screenshot:
https://github.com/NimmakantiDivya/excel-sales-dashboard/blob/main/dashboard.png

Workbook contents:
The Excel workbook includes the following sheets:
1.customer – Customer information
2.orders – Order details
3.products – Product list and pricing
4.Pivot tables – All calculations and summaries
5.Sales_Analysis_Dashboard – Final dashboard

Key Performance Indicators (KPIs):
The dashboard displays:
1.Total Orders
2.Total Revenue
3.Average Customers
4.Order Delivery Time

Dashboard Visuals:
The dashboard contains the following visualizations:
1.Revenue by Category
2.Top 10 Cities by Orders
3.Revenue by Months
4.Revenue by Top 5 Products
5.Revenue by Occasion
6.Revenue by Hour (Order Time)
Slicers for:
   Order Date
   Delivery Date
   Occasion

Tools & Techniques Used:
1.Microsoft Excel
2.Pivot Tables

4.Slicers
5.Data Cleaning
